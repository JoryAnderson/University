1. Install dpkt
	a. Can be done via 'pip3 install dpkt' in terminal OR
	b. Downloading dpkt from the github repository located at:
		https://github.com/kbandla/dpkt
	   Download the source files, go into terminal and run setup.py by typing into console:
			'python3 setup.py install'

2. Run A2 from terminal using the following:
	python3 A2.py <name of capture file>
	(e.g. python3 A2.py pcap)

	(An error will be thrown if the file does not exist, of if there is an incorrect # of 		arguments)
